写前端代码的时候，你有没有遇到过这种情况——

让 AI 帮你写个落地页，代码倒是出来了，配色丑得没法看，字体随便选了个 Arial，布局也是典型的”AI 审美”。你又得花半小时一句句调教：换个配色、字体换成衬线体、CTA 按钮要醒目一点……

UI UX Pro Max（以下简称 UUPM）这个插件想解决的就是这个问题。它本质上是一套给 AI 编程助手准备的 设计知识库 + 推理引擎，让 Claude Code 在写代码之前，先知道该怎么”做设计”。

一、简介

UI UX Pro Max 是一个面向 AI 编程助手的 UI/UX 设计智能技能插件。它内置了 161 条行业推理规则、67 种 UI 风格、161 组配色方案、57 种字体搭配 和 15 种技术栈支持，能够在你用 Claude Code 开发项目时，自动为你生成完整的设计系统并输出高质量的 UI 代码。

简而言之：你只需要用自然语言描述你想要的界面，插件就会帮你搞定设计决策。

十二、常见问题

Q：Python 未安装或报错怎么办？

A：请参考「第二章 环境准备」安装 Python 3.x。安装后用 python3 --version 验证。

Q：插件没有自动激活？

A：确保技能文件存在于 .claude/skills/ui-ux-pro-max/ 目录下。可以尝试重新运行 uipro init --ai claude。

Q：如何确认当前使用了哪个 UI 风格？

A：Claude 在生成代码前通常会输出设计系统的摘要，包括选定的风格、配色和字体。如果没有输出，你可以明确询问：”你使用了什么 UI 风格？”

Q：能否同时使用多种风格？

A：可以。你可以明确指定，例如：”使用 Glassmorphism 作为主风格，结合 Bento Grid 布局。”

Q：如何更新插件？

A：运行 uipro update 或重新执行安装命令即可获取最新版本。

十三、总结

功能 说明
自动激活 在 Claude Code 中直接用自然语言描述 UI 需求即可
设计系统生成 自动生成配色、字体、布局、动效的完整方案
行业智能 161 条推理规则覆盖主流行业，避免设计反模式
多技术栈 支持 15 种前端/移动端框架，针对性编码指南
持久化 通过 Master + Overrides 模式实现设计系统跨会话复用
交付质量 自动执行无障碍、响应式、对比度等交付前检查

一句话总结：告诉 Claude 你想要什么界面，剩下的交给 UI UX Pro Max。
