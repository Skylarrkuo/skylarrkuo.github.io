一、概述

本文将详细解析一个完整的单链表实现程序，该程序包含了单链表的所有基本操作和一些高级操作。代码结构清晰，功能完善，适合数据结构学习和实践。

二、基础定义模块

2.1 宏定义与类型定义

功能说明：

  • 定义了程序中使用的常量和状态码
  • Status 类型用于表示操作是否成功

2.2 链表节点结构

结构说明：

  • data：存储节点数据
  • next：指向下一个节点的指针
  • LNode：节点类型
  • LinkList：链表指针类型（指向头节点）

三、基础操作模块

3.1 初始化链表

功能： 创建一个带头节点的空链表
时间复杂度： O(1)

3.2 创建链表（尾插法）

功能： 使用尾插法创建包含n个节点的链表
优点： 保持输入顺序
时间复杂度： O(n)

3.3 获取链表长度

功能： 遍历链表统计节点个数
时间复杂度： O(n)

四、查找操作模块

4.1 按位置获取元素

功能： 获取第i个位置的元素值
参数： i为位置（从1开始），e用于返回元素值
时间复杂度： O(n)

4.2 按值查找位置

功能： 查找值为e的元素在链表中的位置
返回值： 找到返回位置，未找到返回0
时间复杂度： O(n)

五、修改操作模块

5.1 插入元素

功能： 在第i个位置插入元素e
步骤：

  1. 找到第i-1个节点
  2. 创建新节点
  3. 修改指针完成插入

时间复杂度： O(n)

5.2 删除元素

功能： 删除第i个位置的元素，并返回其值
时间复杂度： O(n)

5.3 清空链表

功能： 删除所有数据节点，保留头节点
时间复杂度： O(n)

5.4 判断链表是否为空

功能： 判断链表是否为空
时间复杂度： O(1)

六、高级操作模块

6.1 链表排序（选择排序）

功能： 使用选择排序算法对链表升序排序
时间复杂度： O(n²)
空间复杂度： O(1)

6.2 链表逆置

功能： 将链表节点顺序反转
方法： 三指针法
时间复杂度： O(n)
空间复杂度： O(1)

6.3 打印链表

功能： 遍历并打印链表所有元素
时间复杂度： O(n)

7.1 菜单显示

7.2 主函数

八、总结

8.1 程序特点

  1. 结构完整：包含单链表的所有基本操作
  2. 用户友好：提供交互式菜单界面
  3. 错误处理：对非法操作进行检查和提示
  4. 内存管理：正确处理动态内存的分配和释放

8.2 时间复杂度总结

操作 时间复杂度
初始化 O(1)
插入/删除 O(n)
查找 O(n)
获取长度 O(n)
排序 O(n²)
